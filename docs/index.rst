**Goodvibes** is an internet radio player for GNU/Linux operating systems. It
strives to be simple and to *just work*.

Under the hood, rock-solid open-source technologies:
`GLib <https://wiki.gnome.org/Projects/GLib>`,
`GStreamer <https://gstreamer.freedesktop.org>` and
`<GTK+ <https://www.gtk.org>`.
